How to Use AlarmTalk

  1. Download and Install: Download the app from GitHub and install the APK on your Android device.
  2. Create an Alarm: Tap the "+" button to create a new alarm and set your desired time.
  3. Customize: Add a title, note, and select your preferred ringtone.
  4. Voice Features: Record a voice message or enable TTS overlay for your alarm.
  5. Set Repeat Days: Choose which days you want the alarm to repeat.
  6. Save and Activate: Save your alarm and toggle it on to activate.
  7. Wake Up: When your alarm triggers, use the snooze or dismiss buttons to manage it.
